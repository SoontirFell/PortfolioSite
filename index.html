My Résumé

Automatrix

Automatrix is a an automation efficiency calculator, loosely based on XKCD 1205. Users can determine if automating a task will save them time based on their estimates of how long the process will take. The formula is made clearly visible to the user and supports sub-calculators to aid in estimation of key variables. Moreover the user has full control over the units of estimation and can even include margins of error, which factor into the outcome, if they desire to.

PlaylistRx

As a music lover and avid Redditor I find a lot of new songs on any number of subreddits. I frequently save these songs using Reddit's built in save feature so that I can go back and listen to them at a more convenient time. PlaylistRx is a tool built to support this habit. With a user's permission PlaylistRx uses OAuth2 to grab all of their saved Spotify and YouTube links and drop them into a table. The user can then choose which songs they wish to save to a Spotify and/or YouTube playlist at the press of the button. When finished the user can also choose to unsave the same or a different set of songs/links from their Reddit saved list. PlaylistRx was originall designed as FireFox WebExtension, but is currently being updated to operate as a stand-alone web page. The original WebExtension was never published, but can be used as a temporary extension. If you'd like to do so download the GitHub repository and follow Mozilla's instructions on setting up a temporary extension.
