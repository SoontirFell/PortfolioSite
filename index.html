Automatrix ~ Git

Automatrix is a an automation efficiency calculator, loosely based upon XKCD 1205. Users can estimate if automating a task will save them time based on their own assumptions. The onscreen formula updates live as the user keys in the necessary variables. The user has the option of to adjust the units and margin of error of a given variable.
